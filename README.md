## About
This mini-project implements an online HTML quiz competition platform. It demonstrates practical usage of semantic HTML, responsive CSS, and vanilla JavaScript for navigation and interactivity. The quiz includes multiple-choice questions, scoring logic, and a registration form to collect participant details.
